# AssetBankPlugin
A plugin for Frosty Toolsuite that adds AssetBank reading support for all major Frostbite titles.

# Supported Animation Codecs
| Name | Support |
| --- | --- |
| RawAnimation | ✅ |
| FrameAnimation | ✅ |
| DctAnimation | ✅ |
| CurveAnimation | ✅ |
| VbrAnimation | ❌ |


# Supported Export Formats
| Name | Extension |
| --- | --- |
| FBX | .fbx |
| SourceMdl Data | .smd |
| GL Transfer Format | .gltf |
